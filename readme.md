# Change Tracking System using SHA256 in Python

Hi there!
I hope you find this repository useful.
This repository is designed for tracking changes in a project using SHA256 hashes and Python.

## Purpose

The primary purpose of this repository is to provide a reliable and secure method for tracking changes in files within a project. By using SHA256, we ensure data integrity, and with Python scripts, we can automate the process of generating and verifying hashes.

## Usage

### Generating SHA256 Hashes

To check for changes in your file system, you can use the provided Python script:

```bash
python check.py <target_directory>
```
